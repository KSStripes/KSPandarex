# KSPandarex

A C++ command-line trading simulation built for Object-Oriented Programming coursework.

## Overview

KSPandarex is a menu-driven cryptocurrency exchange simulator. It reads order-book data from CSV files, lets the user inspect market statistics, place simulated bids and asks, manage a wallet, and move through market timestamps.

The project was created for the University of London CM2005 Object-Oriented Programming module.

## What I Built

- C++ console application with an interactive menu system
- CSV parser for loading order-book data
- Order book model for products, bids, asks, timestamps, and matched sales
- Wallet system for tracking balances and validating whether orders can be fulfilled
- User input flow for entering simulated bids and asks
- Market statistics including min, max, mean open/close, and spread calculations
- Candlestick data generation and terminal-based candlestick display
- Spread statistics with text-based bar chart output
- Xcode project setup for local development

## Tech Stack

- C++
- Standard Template Library
- CSV data processing
- Xcode
